# 🚨 Automated Email Threat Detection System

## 📌 Overview
This project detects phishing and malicious emails by analyzing indicators such as URLs, domains, and IP addresses.

---

## 🛠️ Technologies Used
- n8n (workflow automation)
- VirusTotal API
- WHOIS Lookup
- JavaScript

---

## ⚙️ How It Works
1. Input email data (URL/domain/IP)
2. Send data to VirusTotal API
3. Perform WHOIS lookup
4. Assign risk score:
   - LOW
   - MEDIUM
   - HIGH
5. Send alert via Telegram

---

## 🚨 Features
- Automated threat intelligence enrichment
- Real-time alerting
- Risk scoring system
